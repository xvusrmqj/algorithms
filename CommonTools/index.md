# 计划
本工程的目标是将经常使用的功能整理出来，避免遇到了就忘，并且可以经常回头整理总结。
## 软件设置
1. eclipse设置UTF-8编码：window---preferences---general---workspace---text file encoding---choose utf8.
2. QQ五笔输入法设置：高级设置---勾选中文状态下使用英文标点
